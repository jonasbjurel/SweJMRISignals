JMRI: Sweden 3HMS signalling definition

This JMRI signalling system contains signalling definitions corresponding to the Swedish Railroad rules effective June 1 2019. The purpose has been to cover the Swedish railway signalling system as much as possible, but there are still aspects not covered, and potentially not representing the way the intention was. This signalling system is not in it self tied to any HW - it is left for the user to implement - however there is an compliant implementation for 'Möllehem's Swedish signals found here: XXXXXXXXXX.

This Swedish signal system is based on the Swedish railway authority regulations and guides as per June 1 2019 found here: Regelmoduler i Trafikbestämmelser för järnväg.

Crossing signals:

The swedish signalling system have several masts and aspects for road crossings as well as for water bridge crossings - this includes pre-signalling of the status of crossing guards, immediate at the place status of the crossing guard, as well as the signalling towards road crossing veichles and water passing boats.
